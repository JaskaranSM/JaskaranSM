## 🧭 About

Backend and systems engineer based in **Ontario, Canada**. I've been building things
autonomously since before anyone was paying me to — my instinct in every context is to
find the manual process and automate it out of existence.

I co-authored **python-aria-mirror-bot**, a Telegram automation tool that grew into a
widely adopted open source ecosystem with thousands of users and hundreds of derivative
projects. When its C++ SDK bindings kept crashing, I learned Go, wrote CGO bindings
from scratch, and wrapped them behind a REST service boundary.

I'm looking for a backend or systems engineering role where I can do what I've always
done: **find the broken thing and fix it properly.**
